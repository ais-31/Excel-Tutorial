# Excel Tutorial Project

Repository ini berisi file Excel yang saya gunakan untuk latihan analisis data dan pengolahan spreadsheet.

## Isi File
- `latihan_excel.xlsx` — berisi latihan rumus dasar (SUM, AVERAGE, VLOOKUP, HLOOKUP, IF)
- `dashboard_penjualan.xlsx` — contoh pembuatan dashboard sederhana

## Tujuan
Untuk mendemonstrasikan kemampuan dasar dalam Excel sebagai bagian dari portofolio Data Analyst.
